# 📊 电商数据智能分析平台

基于大模型的电商数据智能分析平台，支持自然语言提问、自动生成 SQL、智能归因分析。

---

## ✨ 功能特性

- 🎯 **意图识别**：精准分类用户问题类型
- 📝 **自然语言转 SQL**：自动将业务问题转化为数据库查询
- 📊 **数据查询**：支持多维度电商数据分析
- 🤖 **AI 归因分析**：深度数据分析和业务归因
- 💬 **交互式对话**：友好的聊天界面

### 支持的分析场景

| 分析类型 | 示例问题 |
|---------|---------|
| 漏斗分析 | 分析引力魔方最近 3 天的漏斗流失情况 |
| ROI 分析 | 查询各渠道的 ROI 排名 |
| 异常检测 | 检查最近 3 天有没有异常数据 |
| 花费分析 | 分析直通车最近 7 天的花费趋势 |
| 点击率分析 | 哪个计划的点击率最低？ |

## 🚀 快速开始

### 环境要求

- Python 3.9+
- 硅基流动 API Key

### 安装步骤

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 创建数据库（首次部署）
python create_ecommerce_ads_db.py

# 3. 配置 API Key
# 创建 .streamlit/secrets.toml 文件，内容如下：
# SILICONFLOW_API_KEY = "sk-your-api-key-here"
# DEFAULT_MODEL = "Qwen/Qwen3.5-9B"

# 4. 启动应用
streamlit run app.py
```

### 访问地址

打开浏览器访问：`http://localhost:8501`

---

## ❓ 常见问题

### Q: 提示"API Key 未配置"
**A**: 确保 `.streamlit/secrets.toml` 文件存在且包含正确的 API Key

### Q: SQL 生成失败
**A**: 检查 API Key 是否正确，网络连接是否正常

### Q: 界面无法访问
**A**: 确保防火墙已开放 8501 端口

### Q: 分析报告不显示
**A**: 检查模型响应是否正常，查看浏览器控制台错误
